# Quickstart - 002 Remediacion Inicial Brownfield

## Verificacion local de esta iniciativa

1. `npm install`
2. `npm run verify`
3. `npm start`
4. En otra terminal: `npm run verify:smoke`

## Recorrido manual esperado

1. `index.html`: login y avance del formulario.
2. `dashboard.html`: acceso solo con PIN valido y render inicial de metricas.
3. `invitacion.html`: lista de nomina sin precargar PIN de toda la plantilla.
