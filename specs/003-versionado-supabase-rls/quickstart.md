# Quickstart - 003 Versionado Supabase y RLS

## Requisitos

1. `SUPABASE_ACCESS_TOKEN`
2. `SUPABASE_DB_PASSWORD`

## Comandos

1. `npm run supabase:check`
2. `npm run supabase:pull`

## Salidas esperadas

- `supabase/config.toml`
- `supabase/migrations/*`
- `supabase/types/database.generated.ts`

## Limite actual

Sin credenciales remotas, este expediente solo puede dejar preparado el flujo y registrar el bloqueo verificable.
