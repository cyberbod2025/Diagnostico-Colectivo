# Quickstart - 004 Ajuste Visual y Movil

## Verificacion local de esta iniciativa

1. `npm install`
2. `npm run verify`
3. `npm start`
4. En otra terminal: `npm run verify:smoke`

## Recorrido manual esperado

1. `index.html`: login, avance del stepper, pasos 2 a 4 y submit.
2. `dashboard.html`: auth, filtros, metricas y listas sin overflow en movil.
3. `invitacion.html`: seleccion de docente, modal PIN e invitaciones imprimibles.
