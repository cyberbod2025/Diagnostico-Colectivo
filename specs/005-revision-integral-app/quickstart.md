# Quickstart - 005 Revision Integral de App

## Verificacion local de esta iniciativa

1. `npm install`
2. `npm run verify`
3. `npm start`
4. En otra terminal: `npm run verify:smoke`

## Recorrido manual esperado

1. `index.html`: login docente, avance por pasos, listado de alumnos y submit.
2. `dashboard.html`: acceso con cuenta autorizada, filtros, metricas y listas.
3. `invitacion.html`: seleccion, revelado de PIN y generacion de invitaciones.
